

✅ 1. 헤더 영역 (<header>)
┌──────────────────────────────┐
│  [로고]        [메뉴 항목들] │  ← 데스크탑 기준: 오른쪽 메뉴 (플렉스 사용)
│              (7개 메뉴)     │
└──────────────────────────────┘


반응형 시 모바일에서는 햄버거 메뉴로 전환

Sticky Header 처리도 가능 (선택)






✅ 2. 히어로 배너 (<section class="hero">)
┌──────────────────────────────┐
│    [배경 이미지 또는 색상]     │
│   "우리 아이 건강 지킴이"      │  ← 큰 문구 (h1 또는 h2)
│   [예약하기 버튼]             │  ← CTA 버튼 (선명한 컬러)
└──────────────────────────────┘


간단한 애니메이션 적용 가능 (페이드인 등)






✅ 3. 병원 소개 요약 (<section class="intro">)
┌──────────────────────────────┐
│ [원장님 이미지]  [간단한 소개 텍스트] │
│                         [→ 자세히 보기] │
└──────────────────────────────┘


2단 구성 (좌: 이미지 / 우: 텍스트)






✅ 4. 진료 과목 안내 (<section class="services">)
┌──────────────────────────────┐
│  [감기] [예방접종] [성장관리] [진료상담]  │ ← 카드 4개 (반응형 그리드)
└──────────────────────────────┘


각 항목은 아이콘 + 텍스트 조합

모바일에서는 2열로 줄이기






✅ 5. 예약 안내 CTA (<section class="cta-reservation">)
┌──────────────────────────────┐
│  [아이콘 또는 이미지]         │
│  "온라인 예약으로 더 편하게"  │
│  [예약하러 가기 버튼]         │
└──────────────────────────────┘


강조 컬러 배경 사용으로 섹션 구분






✅ 6. 병원 둘러보기 프리뷰 (<section class="gallery-preview">)
┌──────────────────────────────┐
│  [사진1] [사진2] [사진3] → (슬라이드 버튼) │
│  [전체 보기 버튼]                      │
└──────────────────────────────┘


슬라이더 or 그리드 택1 가능

클릭 시 전체 갤러리 페이지로 이동




✅ 7. 공지사항 프리뷰 (<section class="notice-preview">)
┌──────────────────────────────┐
│ [● 예방접종 일정 변경 안내]       2025.09.25 │
│ [● 진료 시간 변경 안내]           2025.09.20 │
│ [● 추석 연휴 휴무 안내]           2025.09.10 │
│                         [공지 전체 보기] │
└──────────────────────────────┘


리스트 형식

마우스 오버 시 텍스트 강조





✅ 8. 푸터 (<footer>)
┌──────────────────────────────┐
│  [병원명]  [주소] [전화번호]       │
│  진료시간: 평일 9:00 ~ 18:00       │
│  SNS 아이콘: 인스타 / 카톡 등     │
└──────────────────────────────┘


작은 텍스트 + 컬러 구분 영역
