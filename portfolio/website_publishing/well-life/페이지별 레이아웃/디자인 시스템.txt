


🎨 1. 컬러 시스템 (Color System)
이름	HEX	용도
Primary	#AEE1F9	포인트 컬러 (버튼, 링크 등)
Primary Dark	#7AC5E3	호버 상태
Secondary	#FFF6D6	강조 배경 (진료과목 카드, 섹션 배경 등)
Background	#FFFFFF	기본 배경
Gray Light	#F5F5F5	카드/리스트 배경 보조
Text Black	#333333	제목, 본문 등 기본 텍스트
Text Gray	#666666	서브 텍스트, 날짜 등
Alert / 강조	#FF9E9E	중요 메시지 or 에러 표시

💡 포인트 컬러는 소아과의 부드러운 이미지에 맞춰 밝고 친근한 느낌의 하늘색 계열을 사용합니다.





🔡 2. 폰트 스타일 (Typography)

기본 폰트: Pretendard 또는 SUIT (둘 다 가볍고 가독성 좋음)

용도	사이즈(px)	굵기	라인하이트	예시
H1 (Hero Title)	36px	Bold	1.4	"우리 아이 건강 지킴이"
H2 (섹션 타이틀)	28px	SemiBold	1.4	"병원 소개"
H3 (카드 제목)	20px	Medium	1.4	"예방접종"
본문	16px	Regular	1.6	"소아 감기 진료는 평일 오전 9시부터…"
캡션/날짜	14px	Light	1.4	"2025.09.20"

📦 4. UI 컴포넌트 정의
✅ 카드 컴포넌트 (진료 항목 등)
[아이콘 또는 이미지]
[제목]
[간단 설명]
- 배경: #FFF6D6 또는 #F5F5F5
- 마우스 오버 시 그림자 or 배경 강조




✅ 폼 컴포넌트 (예약하기)

<label>은 항상 <input>과 연결

<fieldset>과 <legend> 사용 권장 (접근성)

포커스 시 테두리 컬러 강조 (#AEE1F9)

버튼: “예약하기”, “초기화” 분리 구성



✅ 탭 / 아코디언 (FAQ)

탭 또는 아코디언은 aria-expanded, role="button" 등 접근성 속성 고려

아이콘: ＋ / － 또는 ▼ 전환



✅ 공지사항 리스트

카드 or 리스트 방식

날짜는 오른쪽 정렬 또는 흐린 색으로 구분

hover 시 제목 컬러 또는 언더라인



✅ 갤러리 썸네일

3~4열 그리드

클릭 시 모달 또는 확대 슬라이드

alt 속성 필수
