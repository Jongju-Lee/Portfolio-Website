H

HTML5

HTML5의 Semantic Tag의 의미를 알고 적극 활용하여 구조적으로 명확하고 접근성이 뛰어난 마크업을 작성합니다. SEO에 친화적인 구조 설계를 통해 검색 엔진 최적화에 기여하고 있고, 웹 표준 및 웹 접근성(WCAG)을 준수하며 UX 향상을 위한 접근성 관련 Attr을 적극 활용합니다.

C

CSS3

CSS3를 기반으로 한 스타일링 작업을 수행하며, Cross-Browsing을 고려해서 안정적인 CSS 속성을 활용하여 작업합니다. 특히 BEM(Block Element Modifier) 방법론을 활용하여 Class Naming Convention을 체계적으로 구성합니다. 이는 팀 단위 협업 시 코드의 가독성을 높이고 유지보수를 수월하게 하며, 대규모 프로젝트에서의 스타일 충돌을 방지하는 데 효과적입니다.

JS

FR

작업에 들어가기 전 벤치마킹 사이트를 물색하고 와이어프레임을 작성한 후, UI 디자인, 기능정의서, HTML구조 설계 순으로 체계적인 퍼블리싱을 합니다. 순차적인 단계를 거침으로써 프로젝트 중 발생하는 불필요한 과정을 줄이고 효율적으로 퍼블리싱을 하고 있습니다.
